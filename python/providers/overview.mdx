## Installation

```bash
# OpenAI
pip install composio-openai

# Anthropic
pip install composio-anthropic

# LangChain
pip install composio-langchain
```

## Usage Pattern

All providers follow a similar pattern:

```python
from composio import Composio
from composio_{provider} import {Provider}Provider

# Initialize with provider
composio = Composio(provider={Provider}Provider())

# Get tools in provider format
tools = composio.tools.get(user_id="default", toolkits=["github"])

# Use with your AI framework
# ...
```
